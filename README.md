## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation & Setup

- Backend
  ```sh
  cd backend
  ```
  ```sh
  npm install
  ```
  ```sh
  npm run start server
  ```
- Frontend
  ```sh
  cd frontend
  ```
  ```sh
  npm install
  ```
  ```sh
  npm run start
  ```
