# heartsinthebyline

A full stack e-commerce application for [heartsinthebyline](https://www.etsy.com/shop/heartsinthebyline).

## Tech Stack

- React
- React Redux
- Axios
- React Styled Components
- React Router
- Express
- MongoDB
- Mongoose
- Stripe API
- Cloudinary
- JSON Web Tokens
- Bcrypt
- Joi
- CSS
- HTML

## Run Memory marker locally

### Backend

1. `cd backend`
2. `npm install`
3. `npx nodemon index.js`

### Frontend

1. `cd frontend`
2. `npm install`
3. `npm run start`
